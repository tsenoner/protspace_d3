ProtSpace Complete Web Components Example

This example demonstrates the integration of all ProtSpace web components:
protspace-data-loader, protspace-scatterplot, protspace-legend, protspace-control-bar, and protspace-structure-viewer.

Try it: Drag & drop an Apache Arrow file (.arrow, .parquet, .feather) or use the sample data below. The visualization will automatically update with your data!
Test files available: basic.arrow (6 proteins) and Pla2g2.arrow (448 proteins) in this directory.
